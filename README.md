연결 리스트 (Linked List)
> 연결 리스트는 데이터를 감싼 노드를 포인터로 연결해서 공간적인 효율성을 극대화시킨 자료 구조이다.
> 삽입과 삭제가 O(1)이 걸리며 탐색에는 O(n)이 걸린다.
